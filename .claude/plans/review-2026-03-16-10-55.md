## [IMPORTANT] N+1 query in document version history

**File:** `backend/src/modules/documents/documents.service.ts:105-120`
**Rule:** I / no N+1 queries in loops

**Issue:** `getVersionHistory` loops through version chain, issuing one DB query per previous version. A document with 10 versions triggers 10 separate queries.

**Current code (lines 109-117):**
```typescript

110 | while (current.previousVersionId) {
111 |     const prev = await this.documentRepository.findOne({
```

**Fix:** Use a recursive CTE:
```typescript

    UNION ALL
    SELECT pd.* FROM property_documents pd
    INNER JOIN chain c ON pd.id = c.previous_version_id
    WHERE pd.company_id = $2
  )
  SELECT * FROM chain ORDER BY version DESC LIMIT 50`,
```
